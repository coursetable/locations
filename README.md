# locations



example run: `python3 walking_matrix_gmaps.py --in buildingCoordinates.json --out walkingETAs.json --resume --block-size 10`

where `buildingCoordinates.json` is a version of [`coursetable/frontend/src/data/buildingCoordinates.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/buildingCoordinates.json) and `walkingETAs.json` will be used in [`coursetable/frontend/src/data/walkingETAs.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/walkingETAs.json).
