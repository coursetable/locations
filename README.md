# locations

where `locations.json` is a version of [`coursetable/frontend/src/data/buildingCoordinates.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/buildingCoordinates.json) and `walkingETAs.json` will be used in [`coursetable/frontend/src/data/walkingETAs.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/walkingETAs.json).
