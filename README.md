# locations



example run: `python3 walking_matrix_gmaps.py --in locations.json --out walking_upper_gmaps.json --resume --block-size 10`

where `locations.json` is a version of [`buildingCoordinates.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/buildingCoordinates.json) and `walking_upper_gmaps.json` is a name for a version of [`walkingETAs.json`](https://github.com/coursetable/coursetable/blob/master/frontend/src/data/walkingETAs.json).
